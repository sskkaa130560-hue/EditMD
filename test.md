- Экспорт закладок из выбранной папки в JSON файл
- Импорт закладок из JSON файла
- Рекурсивная обработка вложенных папок
- Поддержка Firefox и Chrome

## Установка в Chrome

1. Откройте Chrome и перейдите по адресу: `chrome://extensions`
2. Включите "Режим разработчика" (Developer mode) в правом верхнем углу
3. Нажмите кнопку "Распаковать расширение" (Load unpacked)
4. Выберите папку с расширением (папку, где находится manifest.json)
5. Расширение будет установлено и появится в списке

## Установка в Firefox

1. Откройте Firefox и перейдите по адресу: `about:debugging#/runtime/this-firefox`
2. Нажмите кнопку "Загрузить временное дополнение..." (Load Temporary Add-on...)
3. Выберите файл `manifest.json` из папки с расширением
4. Расширение будет установлено и появится в панели инструментов

## Использование

### Экспорт

1. Нажмите кнопку "Экспорт" в расширении
2. В системном диалоге выберите папку с закладками
3. Выберите место на диске для сохранения JSON файла
4. Файл будет создан с именем папки и содержать все закладки рекурсивно

### Импорт

1. Нажмите кнопку "Импорт" в расширении
2. Выберите JSON файл, созданный при экспорте
3. Введите имя папки для импорта (или оставьте по умолчанию)
4. Закладки будут восстановлены в новую папку на панели закладок

## Структура JSON файла

```json
{
  "folderName": "Название папки",
  "bookmarks": [
    {
      "type": "folder",
      "title": "Название вложенной папки",
      "children": [...]
    },
    {
      "type": "bookmark",
      "title": "Название закладки",
      "url": "https://example.com"
    }
  ]
}
```

## Требования

- Chrome 93+ или Firefox 90+
- Поддержка File System Access API (для работы showDirectoryPicker и showSaveFilePicker)
